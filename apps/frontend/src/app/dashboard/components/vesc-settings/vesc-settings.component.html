Configuration Problem

We've detected that your VESC is not properly configured to make use of the Boosted ReVOLT! Dashboard.

Would you like us to automatically configure your VESC for the use of the ReVOLT! Dashboard?

Which settings are changed?

This tool can configure your VESC to interact with the Boosted ReVOLT! dashboard without having to configure your VESC through the official VESC Tool. This tool does not change motor configuration. App configuration that is updated: Controller Id: 22, CAN Status Mode: status messages 1 through 5, Status messages rate: 5Hz, CAN Baud rate: 500k, CAN Mode: VESC, App to use: ADC and UART, ADC control type: current reverse center, ADC min voltage: 1.7V, ADC max voltage: 3.1V, ADC center voltage: 2.3V
